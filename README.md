#Simple PGP UI
Simple front-end for openpgpjs (https://openpgpjs.org/)

To get started run
```
npm install
```

##Source code
All code is in index.html, easily verifiable.
openpgpjs source code is available at https://openpgpjs.org/

###Copy to cordova folder
```
npm run build
```

##Running
With node.js
```
npm run start
```
